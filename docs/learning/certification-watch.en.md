# Monthly certification status watch

Before changing the certification table, check every provider's primary source in the same order. Do not change status from search snippets or training-site summaries alone.

## Check sources

| Provider | Primary source | Check |
|---|---|---|
| AWS | [Coming soon to AWS Certification](https://aws.amazon.com/certification/coming-soon/) | New exams, beta status, renames, and final test dates |
| Azure | [Credential retirement](https://learn.microsoft.com/credentials/support/credential-retirement) | Retirement dates and successor credentials |
| Google Cloud | [Certification catalog](https://cloud.google.com/learn/certification) | Next updates, new credentials, and exam guides |
| Google Cloud | [Exam terms](https://cloud.google.com/certification/terms) | Validity, renewal eligibility, and retake policy |
| OCI | [Oracle Certification](https://education.oracle.com/certification) / [MyLearn](https://mylearn.oracle.com/) | Current-year exam codes, availability, and renewal rules |

## Observation log

### 2026-07-12

- **AWS:** “Coming soon” showed the known SOA-C02 → CloudOps Engineer SOA-C03 update. No additional 2026 change required a site update.
- **Azure:** AZ-204 retires on 2026-07-31 and AZ-500 on 2026-08-31. DP-100 retired on 2026-06-01; AI-102 and AI-900 retired on 2026-06-30, so the site no longer presents them as future events.
- **Google Cloud:** The catalog announced Next '26 updates for the Gemini Enterprise Agent Platform and data/analytics stack. Professional credentials are valid for two years, not three. PDE standard registration is $200 for two hours and renewal is $100 for one hour; both grant two-year validity.
- **OCI:** No `-26` exam code was verifiable from the public certification landing page. Keep the `-25` table until an available `-26` track is confirmed in MyLearn or CertView.

## Update rule

1. Append the date and result to this log.
2. Update the public site only when status, deadline, price, or validity changed.
3. Name a successor only after an official page confirms the credential or course replacement.
4. Change an OCI annual suffix only after verifying an available official exam page—not a search result.
